# hamclock-ng
This repository is an **unofficial, community-maintained continuation** of HamClock by Elwood Charles Downey, WB0OEW (sk).

- Original author: Elwood Charles Downey, WB0EW (sk)
- License: MIT (client-side code only)
- This project does NOT include backend services

## Purpose
This fork focuses on:
- keeping HamClock buildable on modern systems
- fixing bugs and regressions
- small, conservative improvements

This fork depends on backend services provided by the original author.
No backend services are mirrored or replaced.

No commercial use is intended.
